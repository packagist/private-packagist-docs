# Environment-specific setup
## 

This section covers how to configure Private Packagist for the network, authentication, version control, and code hosting platform environments you use.

- [Private networks and firewalls](./private-networks-and-firewalls.md): How to give Private Packagist access to source code stored behind your company firewall.
- [Centralized authentication and SSO](./centralized-authentication-and-sso.md): How to synchronize user access with your code hosting platform to enforce authentication via OAuth (Entra ID, Okta, LDAP, SAML, or OpenID Connect).
- [OAuth integrations](../features/integration-github-bitbucket-gitlab.md): How Private Packagist integrates with GitHub, Bitbucket, GitLab, and other platforms to synchronize users, teams, permissions, and repositories.
